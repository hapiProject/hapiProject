About HAPI

HAPI (Hosptial Application Program Interface) is the latest in Cloud Storage Software that has been specially constructed for the medical industry. With an easy to use login and interface, medicial professionals can create records of patients in a timely manner that is saved to a safe and secure cloud storage unit. Users can quickly request and delete records as well as create suitable outgoing reports for patient families and GPs.
